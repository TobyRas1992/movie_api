Welcome to my movie API reference index!

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

REQUEST URL HTTP METHOD REQUEST DATA FORMAT RESPONSE DATA BODY FORMAT
Get a list of all the user's favorite movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by its title /movies/[movieTitle] GET None A JSON object holding data about the selected movie, including an ID: { id:9, title: "Citizen Kane", director: {name: "Orson Welles", birth-year: "1915", death-year: "1985", bio: "Lorem Ipsum" }}
Get data about the director of a movie by their name /movies/[directorName] GET None A JSON object holding data about the selected director
Get data about a genre (description) by name/title (e.g., "Thriller") /movies/genre/[genreName] GET None A JSON object holding data about all the movies within the selected genre
Add a movie to the database /movies POST A JSON object holding data about the movie to add, structured like: { title: "The Shawshank Redemption", genre. "Drama", director: {"name": "Frank Darabont", "birth-year": "1959", "bio": "Frank Árpád Darabont (born Ferenc Árpád Darabont, January 28, 1959) is..." }} A JSON object holding data about the movie that was added, including an ID: { id:5, title: "The Shawshank Redemption", director: {name: "Frank Darabont", birth-year: "1959", bio: "Frank Árpád Darabont (born Ferenc Árpád Darabont, January 28, 1959) is a..." }}
Add a movie to a user's list of favorite movies users/[userID]//movies/[movieTitle] POST A JSON object holding data about the movie to add, structured like: { title: "The Shawshank Redemption", genre. "Drama", director: {name: "Frank Darabont", birth-year: "1959", bio: "Frank Árpád Darabont (born Ferenc Árpád Darabont, January 28, 1959) is..." }} A JSON object holding data about the movie that was added, including an ID: { id:5, title: "The Shawshank Redemption", director: {name: "Frank Darabont", birth-year: "1959", bio: "Frank Árpád Darabont (born Ferenc Árpád Darabont, January 28, 1959) is a..." }}
Register new user /users POST A JSON object holding data about the user, structured like: {"name": "Peter Jackson", "user_name": "PeterJackson20", "email": "madeupemial@gmail.com"} A JSON object holding data about the user that was added, including an ID: { id:2, username: "PeterJackson20", e-mail: "madeupemial@gmail.com"}
Update an existing user's user name users/[UserID]/[UserName] PUT None A JSON object holding data about the user that was added, including an ID: { id:2, username: "PeterJackson20", e-mail: "madeupemial@gmail.com"}
Deregister an existing user by removing their email (remember to set the Email to "#") /users/[userID]/[userEmail] PUT None A text message indicating whether the email was successfully removed from the profile
Remove a user by ID /users/[userID] DELETE None A text message indicating whether the user was successfully removed
Remove a movie by ID /movies[movieID] DELETE None A text message indicating whether the movie was successfully removed
Delete a movie from a user's list of favorites by ID /users/[userID]/movies/[ID] DELETE None A text message indicating whether the movie was successfully removed
