Welcome to my movie API reference index!

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

REQUEST URL HTTP METHOD REQUEST DATA FORMAT RESPONSE DATA BODY FORMAT
Get a list of all the user's favorite movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by its title /movies/[movieTitle] GET None A JSON object holding data about the selected movie, including an ID: { "_id": "[random_id_number]", "Title": "Citizen Kane", "Description": "The movies centers on XZY", "Genre": "ObjectId([genre_id_number])", "Director": "ObjectId([director_id_number])", "ImageURL": "picture_path.jpg", "Featured": "true"}
Get data about the director of a movie by their name (e.g., "Judd%20Apatow") /directors/[directorName] GET None A JSON object holding data about the selected director
Get data about a genre (description) by name (e.g., "Thriller") /genres/[genreName] GET None A JSON object holding data about all the movies within the selected genre
Add a movie to the database /movies POST A JSON object holding data about the movie to add, structured like: { "Title": "The Shawshank Redemption", "Genre": "ObjectId([id_number])", "Director": "ObjectId([id_number])", "ImageURL": "picture_path.jpg", "Featured": "no"} A JSON object holding data about the movie that was added, including an ID: { _id: [random_id_number], Title: "The Shawshank Redemption", Genre: ObjectId([genre_id_number]), Director: ObjectId([director_id_number]), ImageURL: "picture_path.jpg", Featured: false}
Register new user /users POST A JSON object holding data about the user, structured like: {"Name": "Peter Jackson", "User_Name": "PeterJackson20", "Password": "randompassword", "Email": "madeupemial@gmail.com"} A JSON object holding data about the user that was added, including an ID: { _id: "[random_ind], "Name": "Peter Jackson", "User_Name": "PeterJackson20", "Password": "randompassword", "Email": "madeupemial@gmail.com"}
Add a movie to a user's list of favorite movies users/[userID]/[movieTitle] PUT None A JSON object holding data about the movie that was added to the user's list of favorite movies: { "_id": "[random_id_number]"", "Title": "The Shawshank Redemption", "Genre": ObjectId([genre_id_number]), "Director": ObjectId([director_id_number]), "ImageURL": "picture_path.jpg", "Featured": "true", "Featured_Movies": [ObjectId("1hgh5644566")]}
Remove a movie from a user's list of favorites by ID /users/[userID]/[movieID] PUT None A text message indicating whether the movie was successfully removed
Update an existing user's user name users/[UserID]/[UserName] PUT None A JSON object holding data about the user that was added, including an ID: { _id: "[random_ind], "Name": "Peter Jackson", "User_Name": "PeterJackson20", "Password": "randompassword", "Email": "madeupemial@gmail.com"}
Deregister an existing user by removing their email (remember to set the Email to "#") /users/[userID]/[userEmail] PUT None A text message indicating whether the email was successfully removed from the profile
Remove a user by ID /users/[userID] DELETE None A text message indicating whether the user was successfully removed
Remove a movie by ID /movies/[movieID] DELETE None A text message indicating whether the movie was successfully removed
